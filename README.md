# NAS View

Synology NAS上のメディアをAndroid端末から閲覧するための専用アプリです。

## できること

- NAS上の共有フォルダ一覧を表示
- 選んだ共有フォルダ内をフォルダ単位で閲覧
- 共有フォルダとメディアをタイル表示
- 画像とzipマンガのサムネイル表示
- 現在表示中の画面だけサムネイルを一時保持
- 音声ファイルは親フォルダごとにまとめて表示
- 前回接続先への自動接続
- フォルダ内の名前検索と並び替え
- お気に入り
- ファイル名に含まれるRJコードからDLsite作品名・サークル名を表示
- 共有一覧・フォルダ一覧の手動更新
- 最後に開いたフォルダ、スクロール位置、マンガページ、再生位置の復元
- 画像表示: jpg, jpeg, png, webp, gif, bmp
- 映像再生: mp4, m4v, mkv, webm, 3gp, avi
- 音声再生: mp3, m4a, aac, wav, flac, ogg
- 写真閲覧中の左右スワイプによる前後移動
- 動画をスマホ画面に合わせて自動リサイズ
- 動画再生中のオーバーレイ操作、左右ダブルタップで10秒戻し/送り、前後動画移動
- zipファイルにまとめたマンガのページ送り
- マンガ閲覧中の左右スワイプによるページ移動

## 接続方法

同じWi-FiまたはLAN上にあるSMB対応機器を自動検索できます。見つかった接続候補をタップすると、そのNASを接続先として選択します。

自動検索で見つからない場合は、ホスト名またはIPを手入力できます。

初回接続時だけNASのユーザー名とパスワードを入力します。NASへの接続と認証に成功した場合だけ、Android Keystoreで保護した鍵を使って端末内へ暗号化保存します。次回以降は入力せず自動接続します。

ドメイン、共有フォルダ名、開始フォルダの入力は不要です。

## 動作

接続すると、NAS上で見える共有フォルダ一覧を表示します。`IPC$`、`print$`、名前が `$` で終わる管理用共有は除外します。

共有フォルダをタップすると、まずそのフォルダ直下の内容を表示します。サブフォルダはフォルダタイルとして表示され、タップすると中に入れます。画像とzipマンガはサムネイルを表示します。音声ファイルは曲ごとに並べず、親フォルダ単位でまとめて表示します。

zipマンガを開いたあとは、左スワイプで次ページ、右スワイプで前ページへ移動できます。

## 自動接続

一度接続すると、次回起動時に前回のNASと最後に開いたフォルダへ自動接続します。接続画面の「保存した接続情報を削除」から、接続先と暗号化済み認証情報を削除できます。

## 現在の実装メモ

メディア表示は、表示中フォルダの直下を優先して読み込みます。巨大な写真フォルダでも、共有フォルダ全体を先にスキャンしないため表示待ちを抑えられます。

フォルダの読み込み中に別のフォルダへ移動した場合は、古い読み込みタスクをキャンセルしてSMB接続を閉じ、新しいフォルダの読み込みを優先します。

一度読み込んだフォルダ一覧は最大20フォルダ、15秒間だけメモリに保持します。短時間で元のフォルダへ戻った場合はNASへ再接続せず、保持した一覧をすぐ表示します。

写真などのメディアが大量にあるフォルダでは、最初に60件だけ表示します。下までスクロールすると次の60件を自動で追加します。大量のタイルを一気に作らないことで、メモリ不足を防ぎます。

画像とzipマンガのサムネイルは、現在表示中の画面だけメモリ上に一時保持します。画面を移動すると読み込みタスクをキャンセルしてSMB接続を閉じ、サムネイルも破棄するため、不要な通信や端末ストレージの消費を抑えます。

ファイル名に `RJ` と6～10桁の数字が含まれる場合はRJコードとして検出します。実際に画面へ表示されたタイルだけDLsiteから作品名とサークル名を非同期取得し、成功結果は30日間、情報が見つからなかった結果は1日間端末内へキャッシュします。同時取得は最大2件です。

通常のファイル操作では認証済みSMB接続を再利用し、接続と認証の繰り返しを減らします。キャンセルが必要なフォルダ一覧とサムネイルは専用接続を使用します。

映像と音声は端末へ全体をコピーせず、アプリ内のローカルHTTPプロキシからRange対応でストリーミング再生します。シーク時は必要な位置からNASを読み出します。

接続失敗時は、認証失敗、権限不足、タイムアウト、NASへ接続できない状態を可能な範囲で区別し、再試行ボタンを表示します。

Androidバックアップは無効にしてあり、暗号化済み認証情報を含むアプリデータはバックアップ対象になりません。
